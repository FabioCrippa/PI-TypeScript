Developer Dictionary - JavaScript

Array

O objeto Array é usado para armazenar vários valores em uma única variável.

Neste exemplo abaixo, a variável "cars" armazena os valores da array[] em uma constante

e imprime a lista quando é usado o método getElementById para pegar os valores dentro da array:

const cars = ["Ferrari", "Porshe", "BMW", "Mercedes"];

document.getElementById("vehicle").innerHTML = cars;

String

Uma string JavaScript armazena uma série de caracteres como "Fabio Crippa".

Uma string pode ser qualquer texto entre aspas duplas ou simples:

let carName1 = "HB20";

let carName2 = "Corsa";

document.getElementById("automoveis").innerHTML = carName1 + " " + carName2;

Resultado:

Number

JavaScript tem apenas um tipo de número.

Os números podem ser escritos com ou sem decimais:

let num1 = 3.25;

let num2 = 10;

document.getElementById("numeros").innerHTML = num1 + " " + num2;

Resultado:

Math

O objeto Math permite que você execute tarefas matemáticas.

A matemática não é um construtor. Todas as propriedades/métodos de Math

podem ser chamadas usando Math como um objeto, sem criá-lo:

"buscaPi"

"raizQuadrada"

document.getElementById("buscaPi").innerHTML = Math.PI;

document.getElementById("raizQuadrada").innerHTML = Math.sqrt(16);
