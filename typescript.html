Developer Dictionary - TypeScript

Tipos simples do TypeScript

TypeScript suporta alguns tipos simples (primitivos)

  • boolean - valores verdadeiros ou falsos

    • let vouViajar: boolean = true;

  • number - números inteiros e valores de pontos flutuantes

    • let idade: number = 40;

  • string - valores de texto como "TypeScript Rocks"

    • let firstName: string = "Fabio";

TypeScript Arrays

TypeScript tem uma sintaxe específica para digitar matrizes.

const names: string[];

names.push("Maria");

console.log(names);//["Maria"]

Tipos de objetos TypeScript

TypeScript tem uma sintaxe específica para digitar objetos.

const car: {type: string, model: string, year: number} = {

type: "Fiat",

model: "Ferrari",

year: 2020

};

console.log(car);//{ type: 'Toyota', model: 'Corolla', year: 2009 }

Alias(pseudônimo)

Tipos definidos separadamente das variáveis que os utilizam.

type CarYear = number;
type CarType = string;
type CarModel = string;


type Car = { year: CarYear, type: CarType, model: CarModel };

const carYear: CarYear = 2001;
const carType: CarType = "Toyota";
const carModel: CarModel = "Corolla";


const car: Car = { year: carYear, type: carType, model: carModel };
console.log(car);//{ year: 2001, type: 'Toyota', model: 'Corolla' }

Interfaces

As interfaces são semelhantes aos codinomes de tipo, exceto que só se aplicam a tipos objetos

interface Rectangle { height: number, width: number };

const rectangle: Rectangle = { height: 20, width: 10 };

console.log(rectangle);//{ height: 20, width: 10 }

Ampliação de Interfaces

Estender uma interface significa que você está criando uma nova interface
com as mesmas propriedades do original, além de algo novo.

interface Rectangle { height: number, width: number }

interface ColoredRectangle extends Rectangle { color: string }

const coloredRectangle: ColoredRectangle = { height: 20, width: 10, color: "red" };

console.log(coloredRectangle);//{ height: 20, width: 10, color: 'red' }

Funções do TypeScript

O tipo de valor devolvido pela função pode ser explicitamente definido.

function getTime(): number { return new Date().getTime(); }

console.log(getTime());

Retorno vazio - Void

O tipo pode ser usado para indicar que uma função não retorna nenhum valor (void)

function printHello(): void { console.log('Hello!'); }

printHello();//Hello!

Parâmetros

Os parâmetros de função são digitados com uma sintaxe semelhante às declarações variáveis.

function multiply(a: number, b: number) { return a * b; }

console.log(multiply(2,5))//10

Parâmetros Opcionais - sinal ?

Por padrão, o TypeScript assumirá que todos os parâmetros são necessários, mas eles podem ser explicitamente marcados como opcionais.

function add(a: number, b: number, c?: number) { return a + b + (c || 0); }

console.log(add(2,5))//7

TypeScript Classes - tipos

TypeScript adiciona tipos e modificadores de visibilidade às classes JavaScript.

class Person { name: string; }

const person = new Person(); person.name = "Jane"; console.log(person);//Person { name: 'Jane' }

TypeScript Classes - visibilidade

Existem três modificadores de visibilidade principais no TypeScript.

  • public - (padrão) permite o acesso ao membro da classe de qualquer lugar
  • private - só permite o acesso ao membro da classe de dentro da classe
  • protected - permite o acesso ao membro da classe de si mesmo e a
    todas as classes que o herdam, que está coberto na seção de herança abaixo
class Person { private name: string;

public constructor(name: string) { this.name = name; }

public getName(): string { return this.name; } }

const person = new Person("Jane");

console.log(person.getName());
//person.name não é acessível fora da classe, pois é privado
